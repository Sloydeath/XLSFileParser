Программа реализованна так, что работа идет только с одним файлом (Для простоты демонстрации). Это можно легко изменить, так как я продумал это при
создании архитектуры приложения.
Программа написана с использованием:
 1. Apache POI для доступа к xls-файлам;
 2. Hibernate - для БД.
 3. Клиентская часть реализована с помощью технологии JSP
 Ссылки на используемые библиотеки находятся в файле pom.xml. При использовании Maven библиотеки подгружаются автоматически.
 Они все взяты из репозитория Maven.
